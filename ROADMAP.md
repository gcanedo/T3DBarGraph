# Roadmap

This roadmap tracks likely next steps. It is not a commitment to a specific schedule.

## v0.2.0 - Usability and Data Loading

- Add `LoadFromCSV` helper.
- Add dataset-oriented loading examples.
- Add clearer examples for dynamic updates.
- Add camera presets such as front, side, top, and isometric views.
- Add a `FitToData` helper.
- Improve tooltip readability and placement.

## v0.3.0 - Rendering and Interaction

- Review the current text and axis rendering approach.
- Consider a 2D overlay option for tooltips and labels.
- Improve visual defaults for lighting and materials.
- Add optional value-based color palettes.
- Add image export helpers such as `SaveToPNG`.

## Later

- Split large implementation areas into smaller units.
- Add automated tests where practical.
- Add more complete API documentation.
- Add examples for real dataset interpretation workflows.
- Explore additional chart types only if they fit the component scope.
